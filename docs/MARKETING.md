### Community Marketing

**Goal:** Be present where expats ask questions about inburgering.

**Target communities:**
- [ ] Reddit: r/Netherlands, r/Amsterdam, r/expats, r/dutch
- [ ] Facebook groups: Expats in Amsterdam, Internationals in NL, 30% Ruling groups
- [ ] Internations forums and events
- [ ] LinkedIn expat and relocation communities
- [ ] Telegram/WhatsApp expat groups in Amsterdam, Rotterdam, Den Haag, Eindhoven

**Approach:**
- Provide genuine help answering inburgering questions first
- Share /try link naturally when relevant (not spam)
- Build reputation in communities before promoting
- Track which communities drive the most signups
